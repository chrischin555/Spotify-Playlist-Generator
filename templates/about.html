About Game-Play

What is Game-Play?

Game-Play is an application that utilizes Spotipy, a Python library for Spotify's Web API and OpenAI's API to generate playlists curated for gamers.
It works by taking the title of a game from the user's Steam library (which utilizes Spotify's Steam API for Python)
and takes the information about the game (the name of the game and the genre of the game).
OpenAI uses the name of the game and the genre to find songs that match the vibe of the game.
Users can view details about the song (artist, album, release date) and can listen to a preview of that song.
Users can add the songs they like to a playlist. That playlist can be added to their Spotify account, allowing users to listen to the songs recommended by our application at their convenience.
